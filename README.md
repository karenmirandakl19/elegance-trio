# Elegance Trio Acessórios

Site institucional e vitrine online da loja Elegance Trio Acessórios.

## Funcionalidades
- Site responsivo
- Catálogo de acessórios
- Carrinho de compras em JavaScript
- Finalização de pedido via WhatsApp

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- Bootstrap

## Objetivo
Projeto desenvolvido para estudo e prática de desenvolvimento web e e-commerce simples.
